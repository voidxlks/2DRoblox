# 2D Python Roblox

A simple **2D Roblox-style sandbox game** built with Python using the **Pygame** library.
This project demonstrates basic game mechanics such as player movement, a block world, and a simple game loop.

---

## 🎮 Features

* 2D sandbox environment
* Player movement using keyboard controls
* Block-based ground system
* Simple and beginner-friendly code structure
* Easy to modify and expand

---

## 📂 Project Structure

```
2d-python-roblox/
│
├── main.py        # Main game loop
├── player.py      # Player movement and rendering
├── world.py       # Block world generation
├── blocks.py      # Block definitions (optional)
├── requirements.txt
└── README.md
```

---

## ⚙️ Installation

1. Clone the repository:

```
git clone https://github.com/YOUR-USERNAME/2d-python-roblox.git
```

2. Navigate to the project folder:

```
cd 2d-python-roblox
```

3. Install dependencies:

```
pip install -r requirements.txt
```

---

## ▶️ Running the Game

Run the main file:

```
python main.py
```

A window will open and the game will start.

---

## 🎹 Controls

| Key          | Action     |
| ------------ | ---------- |
| W            | Move Up    |
| A            | Move Left  |
| S            | Move Down  |
| D            | Move Right |
| Close Window | Exit Game  |
